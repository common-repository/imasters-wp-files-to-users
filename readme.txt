== Description ==

iMasters WP Files to users is used to manage and send files to specific users.

== Installation ==

How to install iMasters WP Files to Users

e.g.

1. Upload `imasters-wp-files-to-users` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'iMasters WP Files to Users' menu in WordPress
